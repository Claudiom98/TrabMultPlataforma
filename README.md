# Trabalho Projeto Aplicado de Multilataforma
## 1.Titulo e Descrição do Projeto
 O aplicativo chamado **Aplimpeza** é responsável por tirar uma foto e armazená-la junto com coordenadas geográficas de um lugar que esteja com um acúmulo deplorável de lixo para que voluntários possam executar a limpeza.
