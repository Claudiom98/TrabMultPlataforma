# Trabalho Projeto Aplicado de Multilataforma
## 1.Titulo e Descrição do Projeto
 O aplicativo chamado **Aplimpeza** é responsável por tirar uma foto e armazená-la junto com coordenadas geográficas de um lugar que esteja com um acúmulo deplorável de lixo para que voluntários possam executar a limpeza. Trabalho executado por: **Cláudio Moisés de Oliveira - 2313659**.
 ## 2.Funcionalidades implementadas
 * Sistema de Login (completo)
 * Sistema de cadastro (incompleto, necessita identificar o formato e-mail e implementar sistema de validação de e-mail)
 * Upload de foto, imagem e descrição (completo)
 * Marcar o local como limpo (completo)
 * Consultar localização do local (completo)
