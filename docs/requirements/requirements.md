# Requisitos do Sistema
## Requisitos Funcionais
### RF-01: Autenticação:
* O sistema deve permitir identificar o usuário.
### RF-02: Cadastro:
* O usuário deve criar suas próprias credenciais.
* Cada credencial deve ser única ao usuário.
### RF-03: Reportar Local:
* O usuário deve ser capaz de reportar o local.
* Local deve atrelar uma localização e foto.
### RF-04: Limpar Local:
* O usuário deve poder marcar o local com o status de limpo e tirá-lo da lista
### RF-05: Acessar localização:
* O usuário deve poder acessar as coordenadas de GPS do local sujo.
* Deve ser possível abrir o local em outra aplicação específica para mapas.
## Requisitos Não-Funcionais
### RNF-01: Segurança:
* Senhas são armazenadas de maneira criptografada.
